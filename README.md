## インストール
- 1. npmパッケージをインストール
```
$ npm install
```

- 2. Sketch3.5以下の時**のみ**sketchtoolをインストール
```
$ sudo sh ./install-sketchtool.sh
```

## 使い方
- ``.sketch``ファイルを監視し、画像で書き出す
```
$ npm start
```

- ``.sketch``ファイルを画像で書き出す
```
$ npm run export
```
